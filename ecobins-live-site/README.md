# Eco Bins Website

## Run locally
1. Install Node.js 18+
2. Run:
   npm install
   npm run dev

## Deploy on Vercel
1. Push this folder to GitHub
2. Import the repo into Vercel
3. Add the domain `ecobinsutah.com`
4. Update DNS at your registrar

## Important
The contact form is currently front-end only.
To receive submissions by email, connect a form service such as:
- Formspree
- Resend
- EmailJS
- Web3Forms
